Untitled Live Coding Performance
This live performance featured real-time sound synthesis and audiovisual feedback using code written in Tidal Cycles and Supercollider. I used SuperCollider to generate justly tuned harmonies from digitally processed analog synthesizers and my collaborated Juan Flores resampled my audio feed into TidalCycles to reprocess the sounds into rhythmic glitches. We routed our audio through a TouchDesigner patch that visually transformed the display of the live code driving the performance.
